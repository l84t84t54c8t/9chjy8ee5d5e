  - Update and Install Dependencies: `sudo apt update && sudo apt upgrade -y && sudo apt install -y ffmpeg git python3-pip tmux nano`

  - Clone the Repository: `git clone https://github.com/l84t84t54c8t/9chjy8ee5d5e && cd AlinaMusic`

  - Copy and Edit Environment Variables:

    Copy the sample environment file: `cp sample.env .env`

    Edit the variables in the .env file: `nano .env`

  After editing, press `Ctrl+X`, then `Y`, and press **Enter** to save the changes.
